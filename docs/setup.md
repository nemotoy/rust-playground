# Setup

OS: macOS Catalina
Editor: VSCode

## Install

```sh
# install tool chain
$ brew install rustup
$ rustup-init
$ source "$HOME/.cargo/env"
$ cargo version
cargo 1.71.0

# install lsp
$ brew install rust-analyzer
$ rust-analyzer --version
```
